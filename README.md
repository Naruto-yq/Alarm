# 快捷闹钟
应用简介、联系邮箱、常见问题、隐私政策链接（若有）。支持 URL 填：https://github.com/你的用户名/仓库名

「快捷闹钟」是一款注重清晰与可靠性的闹钟应用：设定简单、到点提醒准确，并可在主界面查看与闹钟相关的天气信息，帮你从容开启每一天。

【核心功能】
• 闹钟与重复：支持每天、工作日、周末、单次及自定义星期；可设置稍后提醒与间隔重复。
• 快捷场景：内置多种常用场景，也可保存自定义场景，一键套用时间与铃声习惯。
• 铃声：内置铃声可选；订阅后可使用自定义音频作为闹钟铃声（支持常见格式，详见应用内说明）。
• 天气与界面：在时钟页展示与闹钟相关的天气信息（需位置权限，仅用于展示天气）。
• 免费使用含广告；订阅后可去除广告并解锁自定义场景与自定义铃声等高级能力。

【订阅说明】
订阅为自动续期项目，费用与周期以购买确认页为准。可在 iPhone「设置」>「Apple ID」>「订阅」中随时取消或管理。
